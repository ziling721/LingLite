# LingLite
包含了上传 审核 识别应用信息等的应用分发平台
本商店的代码为纯ai编写 且部分功能未经过全面的测试（比如不知道是基于前端还是后端解析的图标 未经过实际测试的邮箱发件云控功能 封禁用户上传的功能等） 此项目也只是面向一些不愿意或无能力入门编程的人却还是想拥有一个自己的应用分发平台的项目 起码有80%的概率比这种人瞎捣鼓出来的东西好
关于Android客户端：请使用hankmi应用商店私有部署版本（旧）并按照API.md内的第十条进行使用
关于提示词：请查看linglite.md
关于API文档：请查看API.md
关于零基础部署：请查看deploy.md
以上文件均在linglite.zip内 请自行解压
